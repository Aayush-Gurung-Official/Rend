# Rend

## About
Rend is a Nepal-focused house buy/sell/rent finder project with a React (Vite) frontend and an Express/MongoDB backend API.

## Features
- Frontend built with React + Vite
- Backend REST API built with Express
- MongoDB database via Mongoose
- Typical auth/security dependencies (e.g., `bcryptjs`) and common middleware (`cors`, `morgan`)

## Tech stack
- Frontend: React, Vite, Tailwind CSS
- Backend: Node.js, Express, MongoDB (Mongoose)

## How to run (local)

### Backend
1. Go to `backend/`
2. Install deps: `npm install`
3. Start:
   - Dev: `npm run dev`
   - Prod: `npm start`

### Frontend
1. Go to `frontend/`
2. Install deps: `npm install`
3. Start dev server: `npm run dev`

## Output (what you’ll see)
- Frontend web app UI for browsing/looking up listings.
- Backend API running locally to serve data (and connect to MongoDB if configured).
