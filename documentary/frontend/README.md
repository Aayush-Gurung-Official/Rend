# Frontend Documentation

This folder is reserved for documenting the React + Tailwind frontend of **Rend**.

- Routing structure (`src/routes.js`, pages under `src/pages`)
- UI components (under `src/components`)
- Services for API calls (under `src/services`)
- State management (future work in `src/stores`)
- Styling approach (Tailwind via `styles.css` plus extra CSS in `src/styles`)
